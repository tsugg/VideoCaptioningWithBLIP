## Requirements
* CUDA 11.7
* Python 3.10

## Getting Started 

```sh 
git clone https://github.com/tsugg/VideoCaptioningWithBLIP.git
```

```sh
cd VideoCaptioningWithBLIP
```

```sh
python3 -m venv venv
```

```sh
source venv/bin/activate
```

```sh
pip install -r requirements.txt
```
